About the application for devops

Built for collaboration and version control practice

This static HTML application is structured for a class workflow where every student edits a single profile entry, commits the change, and submits a pull request for review.

The front end displays each student’s name, dream career, city, country of residence, skills, profile photo, and LinkedIn profile.

Student profile fields

  • Full name
  • Dream career
  • City
  • Country of residence
  • Profile picture URL
  • LinkedIn URL
  • Short description
  • Skills list

How students contribute their profile

1

Fork the project

Each student forks the repository to their own GitHub account.

2

Edit one profile object

Update a single student entry in script.js using your real details.

3

Commit changes

Use a clean commit message such as add my student profile.

4

Open a pull request

Submit your PR to the main repository for instructor review.
